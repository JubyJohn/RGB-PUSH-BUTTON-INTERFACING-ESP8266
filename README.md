# RGB-PUSH-BUTTON-INTERFACING-ESP8266

## AIM
To blink different colors of RGB sensor by push button count


## COMPONENTS
1.	ESP8266 NodeMCU
2.	RGB Sensor
3.	Push Button
4.	Jumper Wire
5.	USB cable
